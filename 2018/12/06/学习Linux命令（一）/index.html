学习Linux基本命令

发表于 2018-12-06 更新于 2020-09-23 分类于 Linux 阅读次数：

前言

本文主要介绍常用的 Linux 命令。

Linux系统

  • pwd 打印当前工作目录
  • cd 改变目录
  • ls 列出目录内容
  • file 确定文件类型
  • less 查看文件内容
  • touch 新建文件

操作文件与目录

  • mkdir 创建目录
  • cp 复制文件或目录

cp命令选项

cp在覆盖已存在的文件时默认情况下是 cp -i，即需要用户确认，我们可以这样 \cp 即可无需确认

  • mv 重命名或移动文件和目录

mv命令选项与cp大致相同，mv没有-r选项

  • rm 删除文件或目录

rm命令选项

  • ln 创建硬链接和符号链接

file为相对于sym-link-name的文件，即为相对路径，当然也可以是绝对路径
