当函数参数指定了一种类型之后，无论是传入该类型的包装类型还是基本类型，该函数都可以正常工作。

同样可变参数也会发生函数重载。

当发生函数重载时，编译器会自动匹配应该调用的方法，匹配规则与传入的形参有关，所以当不传入参数时，编译器就无法匹配正确的方法，因而报错。
